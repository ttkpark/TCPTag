# TCPTag

## 프로젝트 소개 및 개발 이유

- C++ 기반 SDL2 라이브러리 게임이다.
- 술래잡기 게임이다.
- 화면을 128X64 크기로 만들었는데, 나중에 만들 게임기에서도 똑같이 보이기 위해 이렇게 작업했다. (게임기는 아직 미완성이다.)

### 역할 소개

- TCPTag PC버젼 코드 설계

## 개발 요소

### #1 SDL2를 활용한 게임 화면 출력

- 게임 엔진에서 화면 버퍼에 도형을 그리고, 그 화면 버퍼를 받아 출력하는 형식으로 화면을 출력했다. 저사양의 프로세서 환경에도 잘 돌아가도록 했다.

### #2 TCP 통신

- 클라이언트와 서버는 서로 TCP 통신을 수행한다. 서로의 위치를 주고받으며, 서로 화면에 상대방이 보이면 상대방 그림을 출력한다.

## 협업 사항

- TCP 프로토콜을 정의하고 적용함.
- 화면 출력방식을 여러 단말기 호환성을 고려함.
